`Spectra` Module
================

The module `pyspectools.spectra` contains most of the high-level interface one
needs to analyze broadband spectra. There are two submodules, `analysis` and
`assignment`, the former contains lower-level functionality for performing
certain analysis routines (e.g. peak finding), while the latter contains classes
that implements most of the user interaction.
